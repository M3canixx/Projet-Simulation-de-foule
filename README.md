Version de UNITY 3D la plus basse utilisé  : 2018.4.9f1.
Membres du groupe ayant travaillé sur le projet : MUBAYA Terry, GHERAB Sabri, EBELING François.
Projet effectué dans le cadre de nos études dans la majeure intelligence artificielle à l’ESME Sudria.
Le projet consiste en une simulation de foule simplifiée dans une pièce fermée se dirigeant vers le même point.
Le principe est de pouvoir générer le nombre de personnes faisant partie de la foule ainsi que la première personne qui se dirigera vers la sortie de la pièce.
Lorsque la scène appelée « finale » est lancée, faire play sur Unity et taper sur la touche "A" pour générer la foule. Taper sur la touche "Espace" afin de générer la première personne en alerte.
